Über das Tool

Das Tool ist ein Hilfsprogramme, welches dir beim Spielen von How To Be A Hero helfen soll. Wir, die Entwickler, stehen in keinem direkten Zusammenhang zu der How to be a Hero e.V. Das Tool ist absolut frei und kann von jedem benutzt werden. Das Tool ist Open Source und kann auf Github eingesehen werden.
